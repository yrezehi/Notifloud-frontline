## Installation

> **_NOTE:_**  The library is not ready or full-featured yet.

### Toast

1 - Refrence `toast.css` to your page
2 - Refrence `toast.js` to your page

### Usage

```
NotifloudToast.success("Success !", "Write success message here");
```
